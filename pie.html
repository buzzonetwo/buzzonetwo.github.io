Pioneers in Engineering UI Development

I worked on the front-end interfaces of the Shepherd (Field Control) project for the Pioneers in Engineering 2018 robotics competition. I was tasked with making simplistic user interfaces that would allow users to easily setup and start matches as well as adjust post match scores. The webpages are in HTML/CSS/Javascript and utilize the Bootstrap, jQuery, and Socket.IO libraries. In order to communicate with the rest of the field control setup, which used Python and LCM as its main communication interface, I wrote a server that used Flask-SocketIO and gevent to asynchronously handle messages from both field control and the webpages.

Additionally, I worked on some aspects of the design of the scoreboard and created the servers that handled communication with the scoreboard and robots.
